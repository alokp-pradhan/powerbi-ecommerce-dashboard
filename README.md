# Ecommerce Analysis Dashboard

## Project Overview
This project is an interactive Power BI dashboard designed to analyze Ecommerce sales data and provide actionable business insights. The dashboard helps track sales performance, customer behavior, product trends, and regional performance through interactive visualizations.

## Tools & Technologies
- Power BI
- Microsoft Excel
- Data Analysis
- Data Visualization

## Key Features
- Sales Performance Analysis
- Regional Sales Breakdown
- Product Category Analysis
- Customer Insights
- Revenue and Profit Tracking
- Interactive Filters and Slicers
- KPI Monitoring Dashboard

## Dashboard Insights
- Identified top-performing products and categories.
- Analyzed sales trends across different regions.
- Tracked revenue and profit growth.
- Evaluated customer purchasing behavior.
